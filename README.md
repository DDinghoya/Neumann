# Neumann
![Icon](https://github.com/turtle-insect/Neumann/blob/master/icon.jpg)

Nintendo Switch SaveDate Backup &amp; Restore Tool

# Build
* Windows10(64bit)
* devkitPro

# Procedure
## pacman
* pacman -Syu
* pacman -Sl dkp-libs
* pacman -S switch-freetype
